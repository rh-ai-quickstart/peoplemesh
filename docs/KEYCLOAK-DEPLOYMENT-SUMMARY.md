# Keycloak Deployment Summary

## What Was Created

A complete Helm chart for **Red Hat build of Keycloak 24.0** with persistent PostgreSQL storage.

### Chart Structure

```
charts/keycloak/
├── Chart.yaml                          # Chart metadata
├── README.md                           # Complete usage guide
├── values.yaml                         # Configuration values
└── templates/
    ├── postgres-secret.yaml            # Database credentials
    ├── postgres-statefulset.yaml       # PostgreSQL with persistent PVC
    ├── postgres-service.yaml           # Database service
    ├── keycloak-cr.yaml                # Keycloak Custom Resource
    ├── keycloak-route.yaml             # OpenShift Route (TLS termination)
    └── realm-import.yaml               # Peoplemesh realm configuration
```

## Key Features

### ✅ Persistent Storage
- **PostgreSQL uses PersistentVolumeClaim** (10Gi by default)
- Data survives pod restarts and redeployments
- NOT ephemeral like the RHBK documentation example

### ✅ Separate Namespace
- Deploys to `samouelian-keycloak` namespace
- Isolated from peoplemesh application
- Can reinstall peoplemesh without affecting Keycloak

### ✅ OpenShift Integration
- Uses OpenShift Route with TLS reencrypt termination
- Auto-generated hostname
- GPU node tolerations included

### ✅ Pre-configured Realm
- Creates `peoplemesh` realm automatically
- Ready for OIDC client configuration
- Includes security best practices

## Deployment Steps

### 1. Prerequisites

```bash
# Verify Keycloak Operator is installed
oc get csv -n samouelian-keycloak | grep rhbk-operator

# Verify namespace exists
oc get namespace samouelian-keycloak
```

### 2. Install Keycloak

```bash
cd /Users/psamouel/Documents/peoplemesh-quickstart

helm install keycloak charts/keycloak/ \
  --namespace samouelian-keycloak \
  --set postgres.password=YOUR_SECURE_PASSWORD
```

### 3. Verify Deployment

```bash
# Check Keycloak status
oc get keycloak keycloak -n samouelian-keycloak

# Check pods
oc get pods -n samouelian-keycloak

# Get Keycloak URL
oc get route keycloak -n samouelian-keycloak -o jsonpath='{.spec.host}'

# Get admin credentials
oc get secret keycloak-initial-admin -n samouelian-keycloak \
  -o jsonpath='{.data.username}' | base64 -d

oc get secret keycloak-initial-admin -n samouelian-keycloak \
  -o jsonpath='{.data.password}' | base64 -d
```

## What's Deployed

### PostgreSQL Database

```
StatefulSet: keycloak-postgres-db
├─ Image: postgres:15
├─ Port: 5432
├─ Database: keycloak
├─ User: keycloak (from Secret)
├─ PersistentVolumeClaim: 10Gi (ReadWriteOnce)
└─ Health Checks: liveness + readiness probes
```

### Keycloak Server

```
Keycloak CR: keycloak
├─ Instances: 1
├─ Database: keycloak-postgres-db
├─ HTTP: Enabled (port 8080)
├─ HTTPS: Enabled (port 8443)
├─ Proxy Headers: xforwarded (OpenShift compatible)
├─ Hostname: Auto-generated by OpenShift
└─ Resources: 1700Mi-2Gi memory, 500m-2 CPU
```

### OpenShift Route

```
Route: keycloak
├─ Target: keycloak-service (Operator-managed)
├─ Port: https (8443)
├─ TLS: reencrypt termination
├─ Hostname: Auto-generated (keycloak-samouelian-keycloak.apps...)
└─ Redirect HTTP→HTTPS: Yes
```

### Peoplemesh Realm

```
KeycloakRealmImport: peoplemesh-realm
├─ Realm ID: peoplemesh
├─ Display Name: Peoplemesh
├─ Registration: Disabled
├─ Email as Username: Enabled
├─ Remember Me: Enabled
├─ Brute Force Protection: Enabled
└─ SSO Session: 30 min idle / 10 hr max
```

## Next Steps

### 1. Access Keycloak Admin Console

```bash
KEYCLOAK_URL="https://$(oc get route keycloak -n samouelian-keycloak -o jsonpath='{.spec.host}')"
echo "Admin Console: $KEYCLOAK_URL/admin"
```

### 2. Create Peoplemesh OIDC Client

In Keycloak Admin Console:
1. Switch to `peoplemesh` realm
2. Create new client:
   - Client ID: `peoplemesh`
   - Client authentication: ON
   - Valid redirect URI: `https://<peoplemesh-route>/api/v1/auth/callback/keycloak`
3. Copy the client secret from Credentials tab

### 3. Create Test User

1. Go to Users → Add user
2. Set username, email, name
3. Set password (non-temporary)

### 4. Get Keycloak Configuration for Peoplemesh

```bash
KEYCLOAK_URL="https://$(oc get route keycloak -n samouelian-keycloak -o jsonpath='{.spec.host}')"

echo "Issuer URL: $KEYCLOAK_URL/realms/peoplemesh"
echo "Discovery: $KEYCLOAK_URL/realms/peoplemesh/.well-known/openid-configuration"
```

## Configuration Reference

### Keycloak Values

| Parameter | Description | Default |
|-----------|-------------|---------|
| `namespace` | Target namespace | `samouelian-keycloak` |
| `postgres.password` | PostgreSQL password | `changeme-keycloak-db` |
| `postgres.persistence.size` | Database storage size | `10Gi` |
| `postgres.persistence.storageClass` | Storage class name | `""` (default) |
| `keycloak.instances` | Number of replicas | `1` |
| `keycloak.resources.requests.memory` | Memory request | `1700Mi` |
| `keycloak.resources.limits.memory` | Memory limit | `2Gi` |
| `realm.enabled` | Create peoplemesh realm | `true` |
| `realm.name` | Realm name | `peoplemesh` |

### Customize Installation

Create `my-keycloak-values.yaml`:

```yaml
postgres:
  password: "MyVerySecurePassword123"
  persistence:
    size: 20Gi

keycloak:
  instances: 2
  resources:
    requests:
      memory: 2Gi
    limits:
      memory: 4Gi
```

Install with custom values:

```bash
helm install keycloak charts/keycloak/ \
  --namespace samouelian-keycloak \
  -f my-keycloak-values.yaml
```

## Troubleshooting

### Keycloak Pod Not Starting

```bash
# Check Keycloak CR status
oc describe keycloak keycloak -n samouelian-keycloak

# Check logs
oc logs -l app.kubernetes.io/name=keycloak -n samouelian-keycloak
```

### Database Connection Issues

```bash
# Test PostgreSQL
oc exec -it keycloak-postgres-db-0 -n samouelian-keycloak -- \
  psql -U keycloak -d keycloak -c "SELECT version();"

# Check database secret
oc get secret keycloak-db-secret -n samouelian-keycloak -o yaml
```

### Realm Not Imported

```bash
# Check realm import status
oc get keycloakrealmimport peoplemesh-realm -n samouelian-keycloak

# Check for errors
oc describe keycloakrealmimport peoplemesh-realm -n samouelian-keycloak
```
